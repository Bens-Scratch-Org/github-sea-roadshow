# Example: Copilot helping write a function

    # Copilot suggests the complete implementation

GitHub Actions 🚀

Automate Your Workflow

CI/CD Made Simple: - Build, test, and deploy from GitHub - Pre-built actions marketplace - Matrix builds for multiple environments - Self-hosted or cloud runners

Popular Use Cases: - Automated testing - Deployment pipelines - Code quality checks - Security scanning

GitHub Actions Example

name: CI/CD Pipeline

on: [push, pull_request]

jobs:
  build:
    runs-on: ubuntu-latest
    steps:
      - uses: actions/checkout@v3
      - name: Run tests
        run: npm test
      - name: Deploy
        if: github.ref == 'refs/heads/main'
        run: npm run deploy
