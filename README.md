## Overview

This repository contains materials and resources for the GitHub South East Asia roadshow events. Our roadshow brings together developers, engineering leaders, and technology enthusiasts to explore modern software development practices, GitHub's powerful features, and industry best practices.

Attendees will learn about:
- GitHub Copilot and AI-powered development
- GitHub Actions for CI/CD automation
- GitHub Advanced Security features
- Best practices for collaboration and code review
- Enterprise-scale development workflows
- Modern DevOps practices

## About the Roadshow

The GitHub South East Asia Roadshow is designed to engage with developers and organizations across the region, showcasing GitHub's latest features, best practices, and developer tools. Join us for hands-on workshops, technical deep-dives, and networking opportunities with fellow developers and GitHub experts.

## Prerequisites

To view and edit the materials in this repository, you'll need:

- **PowerPoint or compatible software** - Microsoft PowerPoint 2016+, Google Slides, or LibreOffice Impress
- **Git** - Version 2.0 or higher
- **Text editor** - For viewing and editing markdown files
- **PDF reader** - For viewing exported presentation materials (optional)

## Getting Started

1. Clone this repository:
   ```bash
   git clone https://github.com/github/sea-roadshow.git
   cd sea-roadshow
   ```

2. Review the presentation materials in the `files/` directory

3. Customize content as needed for specific events

4. For team members: Create a feature branch for your changes
   ```bash
   git checkout -b feature/your-event-customization
   ```
