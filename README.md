# GitHub South East Asia Roadshow

Welcome to the GitHub South East Asia Roadshow repository.

## Overview

This repository contains materials and resources for the GitHub South East Asia roadshow events.

## Contents

- `files/` - Presentation materials and documentation
  - `github-sea-roadshow-presentation.pptx` - Main presentation deck

## About the Roadshow

The GitHub South East Asia Roadshow is designed to engage with developers and organizations across the region, showcasing GitHub's latest features, best practices, and developer tools.

## Getting Started

1. Clone this repository
2. Review the presentation materials in the `files/` directory
3. Customize content as needed for specific events
